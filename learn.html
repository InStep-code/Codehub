HTML Basics

Html is an accronym for HyperText Markup Language , most programmers do not consider Html a programming language, even so html is still the structure of every website.
Lets take real life example of the work html does, when building a house consider html your foundations and bricklayings.

HTML is the structure of your web page.
Html consist of tags like <h1>-<h6>, <p>, <body>, <head>, <html>, <a> and many more, this tags help you define elements on a page.
However htmlhas the habbit of always requesting for a close tag, this is to identify where a particular element defination ended, this is usally recognised with a forward slash /.e.g <h1>Hello World</h1>

Example of html Element Declaration:

<! DOCTYPE html;>
 <html>
   <head>

   </head>
   <body>
       <h1>Hello World</h1>
       <p>I am Learning html basics with CodeHub</p>
   </body>
 </html>

CSS Basics

CSS is an accronym for Casscading StyleSheet, it lets you style your HTML elements with colors, fonts, layouts, and more. Use classes and IDs to target elements in your CSS file.
lets take real life example of css, we've consider html to be our foundation and Building structure, now CSS is the paint, Furniture, Burglaries, Tiles and any other stuffs that beautify our building.
There are basically three ways to style our HTML elements with css

    These are:
  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS: Thids is type of styling applied directly to your html elements, e.g <h1 style="color: red;">Hello world</h1>

Internal CSS: This type of styling is applied at the top of the html document insode the head tag, e.g <head>   <style> Style goes here.... </style>   </head>

External CSS: This type of styling is applied by linking an external file that contains our styling douments, it is linked using this code:
<link rel="stylesheet" href="documentName.css">

Example of Inline CSS Declaration:

<! DOCTYPE html;>
 <html>
   <head>

   </head>
   <body style="background-color: powderblue;">
       <h1 style="background-color: goldenrod;">Hello World</h1>
       <p style="color: purple;">I am Learning html basics with CodeHub</p>
   </body>
 </html>
